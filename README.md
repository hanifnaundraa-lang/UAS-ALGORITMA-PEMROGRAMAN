# Astro Strike

**Astro Strike** is a simple 2D Space Shooter game developed as a Final Project for the Algorithm and Programming course.

## Technology Stack
- **Language:** C++
- **Framework:** Qt 6 (Qt Quick / QML)
- **Build System:** CMake
- **Tools:** VS Code, Qt Creator, Git

## Project Structure
- `src/`: C++ source files for game logic (Controllers, Models, Managers).
- `qml/`: QML files for User Interface (Screens, Components).
- `assets/`: Image and sound resources.
- `data/`: Data storage (e.g., `leaderboard.csv`).
- `docs/`: Project documentation and reports.

## Getting Started
To build the project, use CMake with Qt 6. Open the `CMakeLists.txt` in Qt Creator or use the CMake extension in VS Code.
